# 视频能力验证清单（v0.1）

目标：对视频分析能力做最小可交付验证，统计成功率并记录风险。

## 测试矩阵（5 组）

- 用例 1：短视频（<=30s）+ 有人声 + 室内
- 用例 2：短视频（<=30s）+ 无音轨 + 室外
- 用例 3：中视频（1-3min）+ 背景音乐 + 人物运动
- 用例 4：长视频（>5min）+ 多场景切换
- 用例 5：低清晰度/弱光 + 复杂背景噪声

## 每组必验项

- 能否完成分析（不报错）
- 输出是否为中文
- 时间线是否有事件（至少 3 条，短视频可放宽）
- 是否返回 `recordId`
- 刷新页面后是否仍展示最终结果（不回退进度阶段）
- 重复同参数请求是否命中缓存（响应更快）

## 成功率标准（建议）

- 基础通过线：>= 80%（5 组中至少 4 组成功）
- 推荐发布线：>= 90% 且“长视频 + 低清”至少 1 组成功

## 自动汇总

可用以下命令基于本地持久化记录生成汇总：

```bash
yarn validate:video:report
```

说明：该汇总统计“历史记录中的分析完整度”，用于辅助判断，不替代人工验收。
